                                          ABOUT PROJECT
This is a classification project which uses a patients medical history.



Covid_icu_prediction.ipynb uses patient medical history  to predict whether the patient needs to admitted to an ICU facility or not.


Covid_death_prediction .ipynb uses patient medical history to predict whether the  admitted ICU patient will survive the illness or not.
